createProduct(formData: FormData) — export async function:
1. Call await requireManager()
2. Run createProductSchema.safeParse({ name: formData.get("name"), sku: formData.get("sku"), categoryId: formData.get("categoryId"), reorderThreshold: formData.get("reorderThreshold") })
3. If !parsed.success return { error: "Invalid input. Please check all fields." }
4. Pre-check SKU uniqueness: await prisma.product.findUnique({ where: { sku: parsed.data.sku } }). If found, return { error: "SKU already exists." } — this exact string is displayed as a field-level FormMessage in the UI
5. Validate categoryId refers to an active category: const category = await prisma.category.findUnique({ where: { id: parsed.data.categoryId } }). If !category or !category.isActive, return { error: "Selected category is not available." }
6. Create: await prisma.product.create({ data: { name: parsed.data.name, sku: parsed.data.sku, categoryId: parsed.data.categoryId, reorderThreshold: parsed.data.reorderThreshold } }). Do NOT include currentStock in the create data — @default(0) handles it
7. Call revalidatePath("/products")
8. Return { success: true }

updateProduct(formData: FormData) — export async function:
1. Call await requireManager()
2. Run updateProductSchema.safeParse({ id: formData.get("id"), name: formData.get("name"), sku: formData.get("sku"), categoryId: formData.get("categoryId"), reorderThreshold: formData.get("reorderThreshold") })
3. If !parsed.success return { error: "Invalid input. Please check all fields." }
4. Pre-check SKU uniqueness excluding current product: await prisma.product.findFirst({ where: { sku: parsed.data.sku, NOT: { id: parsed.data.id } } }). If found, return { error: "SKU already exists." }
5. Validate categoryId is active: same pattern as createProduct step 5
6. Update: await prisma.product.update({ where: { id: parsed.data.id }, data: { name: parsed.data.name, sku: parsed.data.sku, categoryId: parsed.data.categoryId, reorderThreshold: parsed.data.reorderThreshold } }). Do NOT include currentStock in the update data — it is managed exclusively by Phase 3
7. Call revalidatePath("/products")
8. Return { success: true }

toggleProductActive(id: string, isActive: boolean) — export async function:
1. Call await requireManager()
2. await prisma.product.update({ where: { id }, data: { isActive } })
3. Call revalidatePath("/products")
4. Return { success: true }

Do NOT call prisma.product.delete() anywhere. Do NOT include currentStock in any schema parse or Prisma write operation in this file.
