createCategory(formData: FormData) — export async function:
1. Call await requireManager()
2. Run createCategorySchema.safeParse({ name: formData.get("name") })
3. If !parsed.success return { error: "Invalid input. Please check all fields." }
4. Pre-check uniqueness with case-insensitive mode: await prisma.category.findFirst({ where: { name: { equals: parsed.data.name, mode: 'insensitive' } } }). If found, return { error: "Category name already exists." } (this exact string — used by the client component for field-level display)
5. Create: await prisma.category.create({ data: { name: parsed.data.name } })
6. Call revalidatePath("/categories")
7. Return { success: true }

updateCategory(formData: FormData) — export async function:
1. Call await requireManager()
2. Run updateCategorySchema.safeParse({ id: formData.get("id"), name: formData.get("name") })
3. If !parsed.success return { error: "Invalid input. Please check all fields." }
4. Pre-check uniqueness excluding current record: await prisma.category.findFirst({ where: { name: { equals: parsed.data.name, mode: 'insensitive' }, NOT: { id: parsed.data.id } } }). If found, return { error: "Category name already exists." }
5. Update: await prisma.category.update({ where: { id: parsed.data.id }, data: { name: parsed.data.name } })
6. Call revalidatePath("/categories")
7. Return { success: true }

toggleCategoryActive(id: string, isActive: boolean) — export async function:
1. Call await requireManager()
2. Update: await prisma.category.update({ where: { id }, data: { isActive } })
3. Call revalidatePath("/categories")
4. Return { success: true }

Do NOT call prisma.category.delete() anywhere in this file — soft-delete only per D-02.
